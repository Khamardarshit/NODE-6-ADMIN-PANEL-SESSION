# Event Management Admin Panel

## 📖 Project Description

The **Event Management Admin Panel** is a full-stack web application built with **Node.js**, **Express.js**, **MongoDB**, and **EJS**, following the **MVC (Model-View-Controller)** architecture. It enables administrators to efficiently manage events through complete **CRUD (Create, Read, Update, Delete)** operations.

The application also supports **image uploads** using **Multer**, making it easy to manage event images alongside event information. This project demonstrates practical backend development, database integration, file handling, and organized application architecture.

# ⚙️ Installation

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Navigate to the Project Folder

```bash
cd project-folder
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure MongoDB

Update your MongoDB connection string in the configuration file.

### 5. Start the Application

```bash
npm start
```

or (for development)

```bash
npm run dev
```

The application will be available at:

```
http://localhost:9000
```
